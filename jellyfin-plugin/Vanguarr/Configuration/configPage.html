Vanguarr

This plugin connects Jellyfin to Vanguarr's recommendations API and creates two native, user-specific Jellyfin views backed by real library items: one for movies and one for shows. Stock Jellyfin does not support adding a brand-new custom home row from a plugin, so Vanguarr surfaces these as first-class browseable views that feel like focused recommendation libraries.

How long Jellyfin should cache each user's resolved Vanguarr suggestions before it refreshes them again.
Maximum number of titles Vanguarr should resolve per user before Jellyfin splits them into the movie and show views.
Display name for the native Jellyfin movie recommendations view. Restart Jellyfin after changing this value.
Display name for the native Jellyfin show recommendations view. Restart Jellyfin after changing this value.
