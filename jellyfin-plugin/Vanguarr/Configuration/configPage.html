Vanguarr

This plugin syncs a per-user Jellyfin playlist from Vanguarr's Suggested for You API. Vanguarr handles ranking and Seerr webhook updates; Jellyfin handles the user-facing playlist.
Base URL of your Vanguarr server, for example http://vanguarr:8000.
Bearer token configured in Vanguarr under Settings -> Integrations -> Suggestions API Key.
How often Jellyfin should refresh user playlists from Vanguarr.
Maximum number of titles to keep in each user's Suggested for You playlist.
Name of the per-user playlist that will appear in Jellyfin.
